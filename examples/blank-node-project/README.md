# Blank Node Project

A basic Express.js server for testing Nexus functionality.

## Quick Start

```bash
cd examples/blank-node-project
npm install
npm start
```

## Endpoints

- `GET /` - Health check
- `GET /health` - Server status
- `GET /api/users` - List users
- `POST /api/users` - Create user
