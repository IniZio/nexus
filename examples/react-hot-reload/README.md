# React Hot Reload Test

A Create React App project for testing Nexus hot reload functionality.

## Quick Start

```bash
cd examples/react-hot-reload
npm install
npm start
```

## Features
- Counter with increment/decrement
- User list fetching from backend
- Real-time title updates
- State management testing
