# Nexus

An AI-native development environment with workspace management and enforcement tools.

## Quick Demo

Run the interactive demo:
```bash
./scripts/demo.sh
```

## Features

- **Workspace Management**: Create, manage, and connect to isolated development workspaces
- **File Synchronization**: Keep files in sync between host and workspace
- **Boulder Enforcement**: Task enforcement with idle detection and mini-workflows
- **IDE Integration**: Plugins for OpenCode, Claude Code, and Cursor
